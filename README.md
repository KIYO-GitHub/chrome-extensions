                                ## 環境構築

                                Chrome拡張機能の設定は以下の通り。

                                ### 1. Zipファイルをダウンロード

                                ### 2. Chromeの拡張機能管理画面を開く

                                Chrome右上の「︙」→「拡張機能」→「拡張機能を管理」

                                ### 3. Zipファイルをドラッグ＆ドロップ

                                1. 管理画面右上の「デベロッパーモード」のスイッチをON
                                2. 2. ダウンロードしたZIPファイルを、そのままブラウザ画面内にドラッグ＆ドロップ
