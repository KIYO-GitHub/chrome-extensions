📖 使い方ガイド — AI Multi Panel

🎯 AI Multi Panel とは？

複数のAIサービス（Gemini・Claude・Genspark・ChatGPT・Copilot・Manus）を1つのウィンドウに並べて同時に使えるChrome拡張です。同じ質問を全AIに一括送信して、回答を比較できます。

⚙️ 基本操作

✏️ 入力ボタン
上部の共有入力エリアを開閉します。テキストを入力して「全パネルに送信」を押すと、すべてのAIパネルに同時に送信されます。
▶ 全パネルに送信
入力したテキストをすべてのiFrameパネルに自動入力します。Ctrl+Enter（Mac: ⌘+Enter）のショートカットも使えます。
⚡ 全パネル実行
各パネルの送信ボタンを自動クリックして、入力済みのプロンプトを一括で実行します。「全パネルに送信」でテキストを入力した後に使います。Ctrl+Shift+Enter のショートカットも使えます。
− / ＋ ボタン
表示するパネル数を1〜4の間で増減します。パネルを追加すると、まだ使っていないサービスが自動で割り当てられます。
レイアウトボタン
パネルの並び方を変えます。2パネルなら「左右」「上下」、4パネルなら「2×2グリッド」「横4列」「縦4列」「メイン+3サブ」から選べます。
パネル内サービス名ボタン
各パネルの左上にあるボタンをクリックすると、そのパネルで表示するAIサービスをいつでも変更できます。同じサービスを複数パネルで使うことも可能です（例：Claude × 2）。
境界線ドラッグ
パネル間の境界線（青い線）をドラッグして、各パネルの表示比率を自由に調整できます。設定は自動保存されます。

🔲 パネル内のコントロール

− / ＋（ズーム）
iFrameの表示倍率を40〜160%の間で調整します。表示が小さすぎる・大きすぎる場合に使用します。
↺（リロード）
そのパネルのiFrameを再読み込みします。ページが固まったときなどに使用します。
↗（新しいタブ）
そのAIサービスを新しいタブで開きます。iFrameで使えないサービスでも通常通り利用できます。
⛶（最大化）
そのパネルを全画面表示します。Escキーで元に戻ります。
✕（パネルを閉じる）
そのパネルを削除します。最低1パネルは必ず残ります。

📋 iFrameに対応していないサービスについて

ChatGPT・Copilot・Manus はセキュリティ上の理由でiFrameへの埋め込みをブロックしています。これらを選択したパネルでは、「全パネルに送信」を押すと、テキストが自動的にクリップボードにコピーされます。

1 入力エリアにテキストを入力
2 「全パネルに送信」をクリック（テキストが自動コピーされます）
3 パネル内の「新しいタブで開く →」ボタンをクリック
4 開いたタブの入力欄で Ctrl+V して貼り付け

💾 設定の保存について

パネル数・レイアウト・各パネルのサービス設定・パネルサイズ比率・入力エリアの高さは、ブラウザのlocalStorageに自動保存されます。次回起動時も同じ状態で再開できます。

⚙️ 管理者設定 — カスタマイズガイド

📁 ファイル構成

📂 ai-multi-panel/（拡張機能フォルダ）
　├ main.js — サービス定義・全ロジック ★ 主な編集対象
　├ main.html — UI骨格（通常変更不要）
　├ main.css — スタイル（見た目の変更）
　├ rules.json — iFrame許可ルール 新サービス追加時
　├ manifest.json — 拡張機能の基本設定
　└ content-scripts/universal-helper.js — テキスト自動入力スクリプト

🔧 サービスのURLを変更する

例：GeminiのURLを変えたい、別のアカウントに切り替えたいなど

// main.js の最上部 SERVICES 配列を編集します const SERVICES = [ { id: 'gemini', name: 'Gemini', url: ''https://gemini.google.com/', ← ここを変更 color: '#4285F4', iframeOk: true }, ... ];
  • url：ブラウザで開くURL（変更するのはここだけでOK）
  • name：パネルヘッダーやドロップダウンに表示される名前
  • color：パネル左端のカラーバーの色（HEXカラーコード）

➕ 新しいAIサービスを追加する

Step 1 ： main.js の SERVICES 配列に追記します

// SERVICES 配列に新しいエントリを追加 { id: 'myai', ← 英数字のID（ユニーク） name: 'My AI', ← 表示名 url: 'https://my-ai.com/chat', ← サービスURL color: '#E91E63', ← カラーコード iframeOk: true }, ← iFrame埋め込み可否（※下記参照）

Step 2 ： iframeOk: true にする場合、rules.json にルールを追加します

// rules.json — 配列の末尾に追加（IDは既存最大値+1） { "id": 9, "priority": 2, "action": { "type": "modifyHeaders", "responseHeaders": [ { "header": "x-frame-options", "operation": "remove" }, { "header": "content-security-policy", "operation": "remove" } ] }, "condition": { "urlFilter": "*://my-ai.com/*", "resourceTypes": ["sub_frame"] } }

Step 3 ： Chrome拡張の管理画面で「再読み込み」ボタンをクリックして変更を反映します

1 chrome://extensions を開く
2 AI Multi Panel のカードにある 🔄 ボタンをクリック
3 AI Multi Panel を開き直す

🎨 見た目をカスタマイズする

背景色・テーマ色
main.css の :root 内の CSS変数を変更します。--color-bg（背景）・--color-accent（アクセント色）など
トップバーの高さ
--topbar-h: 28px の値を変更します
パネルヘッダーの高さ
--panel-header-h: 26px の値を変更します
デフォルトパネル構成
main.js の defaultState() 内の panels 配列を変更します

⚠️ iFrameOk の判定について

サービスがiFrameで表示できるかどうかは、実際にブラウザで確認が必要です。以下の場合は iframeOk: false（クリップボード連携モード）に設定してください：

  • ページが真っ白になる・「埋め込みは許可されていません」と表示される
  • ログイン画面でリダイレクトループが起きる
  • Content-Security-Policy や X-Frame-Options: DENY を強制しているサービス
